# Frontend Rules

- Next.js 16 App Router — read `node_modules/next/dist/docs/` before writing routes
- Tailwind CSS v4 — configured via `postcss.config.mjs`
- Components live in `src/components/`
- Cart state managed via React Context at `src/context/CartContext.tsx`
- Wrap client providers in `src/components/Providers.tsx`
